# GitHub Profile Guide

Use this guide to make the GitHub account look professional for infrastructure, Kubernetes, Nutanix, and AI platform work.

## Profile README Structure

Suggested sections:

- Short headline
- Core focus areas
- Featured projects
- Technical stack
- Current learning or workstream
- Contact or collaboration note

## Example Headline

```md
# Varun Selvaraj

Infrastructure and AI platform engineer focused on Nutanix, Kubernetes, NKP, NAI, GPU workloads, and deployment automation.
```
